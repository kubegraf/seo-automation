# Kubegraf — AI Kubernetes SRE Platform

Kubegraf receives alerts from Prometheus, Grafana, and Slack, automatically investigates Kubernetes workloads using AI, performs root cause analysis, and applies SafeFix remediations.

## How It Works

1. **Alert Ingestion** — Receives alerts from Prometheus AlertManager, Grafana, or Slack
2. **AI Investigation** — Automatically queries pod logs, events, metrics, and deployment history
3. **Root Cause Analysis** — AI identifies the root cause with confidence score
4. **SafeFix** — Suggests or automatically applies the appropriate fix
5. **Approval Gate** — Human approval required before applying fixes (optional)
